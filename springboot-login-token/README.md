# SpringBoot+JWT 实现无状态登录 

## 介绍
- 本项目是一个基于 SpringBoot+JWT 实现无状态登录的简易 Demo，主要用于学习无状态登录的思想和 Json Web Token 的使用
- 持久层框架使用的是 [MyBatis]()
- 不涉及其他任何安全校验框架（Shiro、SpringSecurity等）
- 不涉及前端部分，接口测试使用的是 [Postman](https://www.postman.com/)
